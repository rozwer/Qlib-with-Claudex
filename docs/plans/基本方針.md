# Qlib-with-Claudex / RD-Agent-with-Claudex Design

## Overview

Microsoft Qlib と RD-Agent をフォークし、OpenAI API 依存を Claude Code + サブエージェント + Codex に置き換えた OSS として再配布する。

## Project Structure

| リポジトリ | ソース | 内容 |
|---|---|---|
| Qlib-with-Claudex | microsoft/qlib | 量的投資フレームワーク本体 |
| RD-Agent-with-Claudex | microsoft/RD-Agent | LLM駆動の研究開発エージェント |

## Design Decisions

- **AI基盤**: Claude Code（セッションベース）+ サブエージェント + Codex
- **API方式**: OpenAI API呼び出しではなく、VSCode上のClaude Codeセッションから操作
- **ターゲット**: クオンツ開発者（Python + VSCode）
- **ライセンス**: MIT継承
- **構成**: 2リポ独立、統一ブランド `with-Claudex`

## Phase 1: Initial Scope

1. 両リポをクローン
2. OpenAI API 依存箇所の洗い出し（grep ベース）
3. 依存レポート作成

## Phase 2 (Future)

- OpenAI API 呼び出しを Claude Code スキル / サブエージェント / Codex に置換
- CLAUDE.md / スキル / フック整備
- ワークフロー層の再設計
- ドキュメント整備・公開
